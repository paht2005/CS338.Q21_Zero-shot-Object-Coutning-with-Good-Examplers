### Experiment exp2

This folder stores one set of VA-Count experiments used in the project.

Typical contents:

- `data/` – local copy or subset of FSC147 used in this run
- `GroundingDINO/` – GroundingDINO code and weights
- `notebook/` – Jupyter notebooks for exploratory analysis or debugging
- `util/` – small helper scripts specific to this experiment
- `visualizations/` – qualitative visualizations (density maps, predictions)
- `wandb/` – Weights & Biases logs for training and evaluation

This folder is mainly for **reproducibility and debugging**.  
To run the main code, use the project under `code/cs331-source-code` instead.
