# `images/`

Static image assets used in the GitHub repository README and documentation.

## Files

| File | Used in | Description |
|------|---------|-------------|
| `thumbnail.png` | Root `README.md` | Project thumbnail / banner image |

## Adding new images

Place screenshots, architecture diagrams, or demo GIFs here and reference them
from Markdown files using relative paths:

```markdown
![Description](images/your-image.png)
```
